# FoodMarketProject2026
## Project Overview
FoodMarketProject2026 is a Salesforce-based food marketplace application designed to connect home chefs, restaurants, and customers through Experience Cloud.

This project includes:
- Salesforce Sales Cloud
- Experience Cloud
- Apex
- Lightning Web Components (LWC)
- Flow Automation
- Marketing Automation
- Agentforce AI
- Data Cloud concepts

---

## Features

### Customer Features
- Browse meals
- Search and filter recipes
- Place food orders
- Rate and review meals

### Chef Features
- Manage recipes
- Receive order notifications
- Track customer reviews

### Admin Features
- Order management
- Campaign management
- Dashboards and reports
- AI-powered automation
